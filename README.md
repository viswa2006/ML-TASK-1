# Customer Churn Prediction using Machine Learning

This project uses Machine Learning to predict whether a customer is likely to leave a company's service (customer churn). The notebook includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation. Customer information is cleaned, encoded, and used to train a Random Forest Classifier that predicts churn based on various customer attributes. The project is implemented in Python using Jupyter Notebook and libraries such as Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn. The trained model is evaluated using accuracy score to measure its prediction performance.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithm
- Random Forest Classifier

## Features
- Data preprocessing and cleaning
- Handling missing values
- Exploratory Data Analysis (EDA)
- Label Encoding for categorical features
- Model training and testing
- Customer churn prediction
- Model evaluation using accuracy score

## How to Run
1. Install the required libraries:
```bash
pip install numpy pandas matplotlib seaborn scikit-learn
```

2. Open the notebook:
```bash
jupyter notebook ML-1.ipynb
```

3. Run all the cells in sequence.

## Output
The model predicts whether a customer will churn or not and displays the accuracy of the Random Forest model on the test dataset.
